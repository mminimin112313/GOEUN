# 계정 초기화 (Account Reset) 사양서

## 1. 개요
사용자가 자신의 학습 데이터를 모두 지우고 처음부터 다시 시작할 수 있는 기능을 제공한다. 실수로 인한 초기화를 방지하기 위해 명시적인 텍스트 확인 절차를 거친다.

## 2. 초기화 대상 데이터
- **프로필 정보**: 레벨(1), XP(0), 코인(0), 스트릭(0)
- **미션 상태**: 일일/주간/튜토리얼 미션 진행도 초기화
- **학습 기록**: 퀴즈 히스토리(quizHistory) 전체 삭제
- **오답 데이터**: 오답 노트(wrongNotes), 문제별 메모(questionMemos), 복습 기록
- **진행 데이터**: 이미 본 문제 ID(seenIds) 초기화

## 3. UI/UX 흐름
1. **버튼 배치**: `Settings` 페이지 하단의 'Danger Zone' 섹션에 '계정 리셋' 버튼 배치.
2. **확인 모달**: 버튼 클릭 시 모달 팝업.
   - 문구: "모든 데이터가 영구적으로 삭제됩니다. 계속하시겠습니까?"
   - 확인 절차: 입력창에 `"리셋을 원합니다"`를 정확히 입력해야 함.
3. **실행**: 입력이 일치하면 '영구 삭제 및 초기화' 버튼 활성화.
4. **완료**: 초기화 완료 후 홈(Dashboard)으로 이동 및 성공 알림.

## 4. 기술적 고려사항
- `synced` 스토어는 Firestore와 LocalStorage 양쪽을 초기화해야 함.
- 초기화 후 애플리케이션 상태를 즉시 반영하기 위해 Svelte 스토어의 `set` 메서드 활용.
