# ADR: 퀴즈 메모 및 상호작용 개선

## 상태
승인됨 (Approved)

## 배경
기존 퀴즈 앱의 메모 모드는 다음과 같은 한계가 있었습니다:
1. 메모 모드 활성화 시 일반적인 터치 기반 스크롤이 불가능함.
2. 선택지 클릭 시 미세한 움직임이 드로잉으로 인식되어 클릭(정답 선택)이 무시됨.
3. 드로잉 선이 단순 직선 연결로 되어 있어 시각적으로 끊겨 보임.

## 결정 사항
참조용 앱의 `DrawingCanvas` 로직을 이식하여 다음 기능을 도입합니다:

1. **멀티터치 스크롤 (Multi-touch Scroll)**: 
   - 두 손가락 이상 터치 시 드로잉을 즉시 중단합니다.
   - Pointer 이벤트의 이동 거리를 계산하여 수동으로 `window.scrollBy`를 호출, 부드러운 스크롤을 제공합니다.

2. **스마트 탭 (Smart Tap)**:
   - 터치 시작부터 종료까지의 시간(< 500ms)과 거리(< 15px)를 측정합니다.
   - 조건 만족 시 '탭'으로 간주하여 캔버스 레이어를 투과해 하단 요소(`button`, `input` 등)에 프로그래밍 방식의 클릭 이벤트를 전달합니다.

3. **고급 드로잉 (Quadratic Curve)**:
   - 단순 직선 대신 Quadratic Curve를 사용하여 선을 부드럽게 연결합니다.
   - `isDrawing` engagement 임계값(10px)을 설정하여 안정성을 높입니다.

4. **전용 스크롤 존 (Scroll Zone)**:
   - 화면 우측 12px 영역에 시각적 가이드를 제공하고, 해당 영역 터치 시 드로잉을 원천 차단하여 스크롤 전용으로 사용합니다.

## 기대 효과
- 사용자는 메모 중에도 자유롭게 화면을 이동할 수 있습니다.
- 펜이나 손가락으로 문제를 풀 때 선택지 클릭의 정확도가 향상됩니다.
- 메모 선이 더 자연스럽고 매끄럽게 표현됩니다.
